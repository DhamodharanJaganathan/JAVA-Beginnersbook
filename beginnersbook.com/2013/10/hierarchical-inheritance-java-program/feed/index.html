If you call methodA() on myclass object it wont work since it does not have have access to that method but if we extend method B and have the same method within myclass then this would be method overloading…

class A
{
public void methodA()
{
System.out.println(“method of Class A”);
}
}
class B extends A
{
public void methodB()
{
System.out.println(“method of Class B”);
}
}
class C extends A
{
public void methodC()
{
System.out.println(“method of Class C”);
}
}
class D extends A
{
public void methodD()
{
System.out.println(“method of Class D”);
}
}
class MyClass extends B
{
public void methodB()
{
System.out.println(“method of Class B”);
}
public static void main(String args[])
{
B obj1 = new B();
C obj2 = new C();
D obj3 = new D();
obj1.methodA();
obj2.methodA();
obj3.methodA();

MyClass a = new MyClass();
a.methodA();
a.methodB();
}
}

Output:
method of Class A
method of Class A
method of Class A
method of Class A
method of Class B
