How to get Bing Search API key in order to get LSI – SEOPressor Plugin

By Chaitanya Singh | Filed Under: SEOPressor V5

If you have purchased one of the best plugin  SeoPressor V5 recently and not sure why the LSI feature of the plugin is not working then follow this guide. You will not be able to browse LSI keywords until you setup an API for this. Unfortunately, this plugin doesn’t come up with inbuilt LSI API, you have to configure it to do so. Its quite simple, just follow the below steps to get it done-

Recommended readings: SEOPressor v5 review.

Step 1: Login to your Microsoft account here. If you don’t have one then signup, it’s completely free.

Step 2: Visit this link for Bing Search API. You would find a screen like below for Bing Search API pricing-

As shown in the above screenshot, signup for free (5000/month transactions), fill the required details and complete the registration process. Once you are done with this, click on “My Account” and click “Account info” OR “Account keys”.

Step 3: Copy the key either from account information tab or else visit Account keys tab to get the key. We would need to provide this key in plugin settings to configure the bing API with the plugin.

Step 4:  Paste the key at the specified field, like I did below. Validate the API key and once it shows success message, save the settings. That’s it, you have successfully configured the settings in order to get LSI keywords.

In order to test whether it’s working fine, goto WordPress dashboard and create a new post or edit and existing post. Give the target keyword at specified field and the plugin will show the LSI automatically (relevant to the keyword specified by you).

If you are a newbie and not sure what is LSI and how it can be useful for search engine optimization then read this post – What are LSI keywords and how to get them using SeoPressor.

Documentation: Bing API Documentation.
