Why to use LSI?
In order to rank high in Search engine such as Google, it is good to have more and more LSI terms (or keywords) related to targeted keywords. Read more at What are LSI keywords and how to improve SEO using them.

How to find them?
1) Search engine suggestion
2) Google related keywords at the bottom of the search results for main keyword.
3) Google Adwords tool
